Drum Kit project -- to play different drums sound on mouse click event and keyboard press event..

Selecting all drums using querySelectorAll() with class drum and adding addEventListener on mouse click
and based on the this.innerHTML creating animation and making sound

adding addEventListener on whole document for keypress or keydown and based on parameter of callback
function getting event.key to know which key is pressed

To play particular sound :
Inside switch case creating object of Audio class and calling .play() method to play sounds

For animation :
using classList.add() to add pressed  class and using setTimeout to remove this class after a
certain time to create animation.
